แก้ไขชื่อทีมแบบ JSON

พิมพ์ชื่อทีมในรูปแบบ JSON array เช่น ["ทีม A","ทีม B","ทีม C"] — ใช้สำหรับคัดลอง/วางชื่อทีมจำนวนมากได้ง่าย

คำแนะนำการใช้งาน

• ใช้งาน: พิมพ์ชื่อทีมในรูป JSON array แล้วกด "ใช้งาน"

• โหลด: กด "โหลดจากทีมปัจจุบัน" เพื่อดึงชื่อทีมที่มีอยู่มาแก้ไข

• ดาวน์โหลด: กด "ดาวน์โหลดไฟล์" เพื่อบันทึกเป็นไฟล์ .json

• รูปแบบ: ["ชื่อทีม 1","ชื่อทีม 2",...]
